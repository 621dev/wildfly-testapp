WildFly Maven Application

Jakarta EE 10 • Java 17 • Maven Setup Complete

프로젝트 뼈대 생성 완료

현재 리포지터리 내에 표준 Maven 프로젝트의 폴더 구조와 WildFly 및 Jakarta EE 10 기반의 애플리케이션 개발에 필요한 pom.xml, web.xml, beans.xml 및 기본 Servlet 코드가 모두 자동으로 생성되었습니다.

1

Maven 설치

시스템에 Maven이 설치되어 있어야 합니다. CLI 혹은 IDE(IntelliJ, Eclipse, VS Code 등)에 내장된 Maven을 이용할 수도 있습니다.

2

서버 연결 & 배포

WildFly 서버가 구동 중인 상태에서 mvn wildfly:deploy 명령어를 실행하면, 작성한 웹 애플리케이션이 자동으로 빌드 및 배포됩니다.
